## v2.4（2026-07-21）— 财务穿透+段永平裁决正式化

### Phase 2.5: 财务穿透标准流程
4步验证链（必须在资金面信号之前执行）：
1. income: revenue_yoy / grossprofit_margin趋势 / netprofit_margin
2. fina_indicator: ROE(>15%) / FCFF(正) / OCF/营收(>15%) / q_profit_yoy / debt_to_assets
3. forecast: 业绩预告type / 增速范围 / 无预告=⚠️
4. report_rc: ⚠️时效陷阱！可能只返回旧报告。检查report_date。

判定矩阵：
| 信号 | 判定 | 案例 |
|------|------|------|
| ROE>15% + FCFF正 + 利润正增长 | ✅ 健康 | 东方电缆 |
| ROE>15% + FCFF正 + 利润暴跌但H1反转 | ⚠️ 待确认 | 生益科技 |
| 利润负增长 + FCFF负 + 趋势恶化 | 🔴 恶化 | 汇川/贝岭 |
| 利润暴跌80% + 毛利<30% + 无定价权 | 🔴🔴 价值陷阱 | 上海贝岭 |

### Phase 2 Step 8: 段永平裁决（正式化）
- 段永平≥7 → 检查巴菲特+李录 → ✅
- 段永平<7 → ❌ 价格不对不买（即使总分最高）
- 实战：生益7.5最高但段5→❌；东方电缆7.25段8→✅
